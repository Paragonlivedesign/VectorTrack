## VectorTrack 0.5.0 beta

### VectorTrack (desktop)

- Open Files and project summaries show **project names** instead of project numbers
- **Project numbers are optional** when creating projects (name is required)
- Automatic Vectorworks file and idle tracking, session explorer, PDF reports, light/dark theme
- Optional cross-machine log sync via a cloud-synced folder

### VectorTrackScript (plug-in)

- Version aligned to **0.5.0 beta**
- In-Vectorworks time summary for the open document

### Install

Download **VectorTrack-0.5.0-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.

Windows may show a SmartScreen warning — the installer is not code-signed yet. Choose **More info → Run anyway**.

Full changelog: https://github.com/Paragonlivedesign/VectorTrack/blob/main/VectorTrack%200.5/CHANGELOG.md
