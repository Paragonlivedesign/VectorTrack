## VectorTrack 0.5.1 beta

### Fixes

- **Logs and reports in AppData** — installed app no longer fails to start or export when running from Program Files
- **Single-instance guard** — second launch focuses the existing window

### New and improved

- **Vectorworks setup prompts** when auto-detect fails or succeeds
- **Log Library** shows auto-linked Vectorworks Log.txt paths
- **Idle detection settings** with optional bypass rules
- **Reports** — Master Summary, Project Detail, and Client Statement
- Updated app icons

### Install

Download **VectorTrack-0.5.1-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.

Windows may show a SmartScreen warning — the installer is not code-signed yet. Choose **More info → Run anyway**.

Full changelog: https://github.com/Paragonlivedesign/VectorTrack/blob/main/VectorTrack%200.5/CHANGELOG.md
