## VectorTrack 0.5.2 beta

### Fixes

- **Crash stability** — hotkeys and idle detection no longer fight over the keyboard listener; UI updates from hotkeys run on the main thread
- **Status bar** — version (e.g. `0.5.2 beta`) shown bottom-left

### Install

Download **VectorTrack-0.5.2-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.

Windows may show a SmartScreen warning — the installer is not code-signed yet. Choose **More info → Run anyway**.

Full changelog: https://github.com/Paragonlivedesign/VectorTrack/blob/main/VectorTrack%200.5/CHANGELOG.md
