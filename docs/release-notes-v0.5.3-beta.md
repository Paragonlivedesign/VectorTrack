## VectorTrack 0.5.3 beta

### Fix

- **Crash when Vectorworks is open** — VectorTrack no longer walks every child window inside Vectorworks (which could freeze or crash the app, especially on VW 2026 startup)

### Install

Download **VectorTrack-0.5.3-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.
