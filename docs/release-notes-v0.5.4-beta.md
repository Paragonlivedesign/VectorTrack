## VectorTrack 0.5.4 beta

### New and improved

- **Notifications** use the system tray (no console flash)
- **Orphan projects** from file assignments auto-register in Project Editor
- **Open Files** — edit project on double-click; click active row Status/Live to resume tracking
- **Pause/Resume** menu label and idle-gap fix when resuming

### Install

Download **VectorTrack-0.5.4-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.

Full changelog: https://github.com/Paragonlivedesign/VectorTrack/blob/main/VectorTrack%200.5/CHANGELOG.md
