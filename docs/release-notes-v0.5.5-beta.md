## VectorTrack 0.5.5 beta

### New

- **Check for Updates** (Help menu) — compares your build to GitHub Releases
- **Tray icons** reflect tracking state; pause/resume from tray menu
- **Cross-machine sync** — share log snapshots and project assignments via your cloud sync folder
- **Stable Machine ID** from Vectorworks install data (when available)
- **History** — Machine + Status columns, conflict/excluded highlighting, 30-day default range
- **Log setup prompt** when Vectorworks Log.txt is not found

### Install

Download **VectorTrack-0.5.5-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.

Full changelog: https://github.com/Paragonlivedesign/VectorTrack/blob/main/VectorTrack%200.5/CHANGELOG.md
