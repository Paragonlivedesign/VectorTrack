## VectorTrack 0.5.7 beta

### Hotfix

- **Startup crash** — fixed crash on launch when catalog sync ran before the UI was ready

### New and improved

- **Catalog sync** — share clients/projects across machines; review and resolve conflicts
- **Project budgets** — hours or money (not both), with progress on the Projects tab
- **Vectorworks** — better install detection and auto-select running VW
- **UI** — DPI-aware sizing and compact layouts

### Install

Download **VectorTrack-0.5.7-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.

Full changelog: https://github.com/Paragonlivedesign/VectorTrack/blob/main/VectorTrack%200.5/CHANGELOG.md
