## VectorTrack 0.5.9 beta

### Highlights

- **Shared core** — desktop and plug-in now use the same log/sync/identity code (`vectortrack-core`)
- **Refactor** — cleaner internals (sync coordinator, typed settings, slimmer main window)
- **Quality** — GitHub Actions CI and expanded test coverage

### Install

Download **VectorTrack-0.5.9-Setup.exe** (desktop) and **VectorTrackScript_0.5.zip** (Vectorworks plug-in) from the assets below.
